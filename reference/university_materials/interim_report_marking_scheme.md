## Quick Checklist for a Strong Interim Report

To target a high mark, the report should:

- Provide a clear and well-cited project context.
- Explain a convincing motivation for the project.
- State one concise high-level aim.
- Present professional, measurable, and aim-aligned objectives.
- Describe progress clearly, including work completed, tools, papers, datasets, and design decisions.
- Justify major project choices rather than only listing them.
- Include a realistic project plan with tasks, timeline, milestones, and links to objectives.
- Provide a detailed data management plan covering data collection, storage, documentation, security, ethics, legal compliance, sharing, and preservation.
- Use professional technical writing throughout.
- Present references consistently using an appropriate citation style.
